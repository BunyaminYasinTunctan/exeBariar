#exeBarrier
An application that secures your applications and files with password protection.

Note: Converting the application into an .exe file will provide a more convenient and user-friendly experience.

Operating Steps:
Step 1: Launch the application.

Step 2: Select your language (TR/EN).

Step 3: Select the .exe file you wish to encrypt and enter the password.

Password Requirements:
To ensure security, your password must meet the following criteria:

Length: Minimum of 12 characters.

Case Sensitivity: Must include at least one uppercase letter and one lowercase letter.

Digits: Must contain at least one numerical digit.

Special Characters: Must include at least one special character (e.g., . , ! @ # $ % ^ & *).
